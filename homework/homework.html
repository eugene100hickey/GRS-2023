You are tasked with reproducing the following figure:

Procedure

  • You’ll need the tidyverse library, the carData library (install from CRAN by install.packages(“carData”)), the scales package, the showtext package, and the ggthemes library (install from CRAN by install.packages(“ggthemes”))
  • The dataset is called UN98 from carData.
  • There are three aesthetics; for x = lifeFemale, y = GDPperCapita, and colour = economicActivityFemale
  • Add the graph title, the legend title, and add a caption
  • The y axis should be plotted on a log scale with the labels set to dollar() from the scales package
  • Facet by region, use either facet_grid() or facet_wrap() as appropriate
  • Add a geom_smooth() with method = “lm” and se = F
  • Add an appropriate colour scale for the economicActivityFemale aesthetic.
  • Set the theme for the plot with the theme_wsj theme from the ggthemes package (wsj copies the theme from the Wall Street Journal).
  • install some handwriting font with font_add_google(), don’t forget to make it available with showtext_auto(). Use theme(strip.text = element_text(…….)) to print the continent names in this font. Make sure you do this after theme_wsj() or else it gets overwritten.
  • As before, add something extra that enhances the image.
